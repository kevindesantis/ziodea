# Deacon Biamonte S.p.A.

Sito statico personale in HTML, CSS e JavaScript, pronto per GitHub Pages.

## Come caricarlo su GitHub

1. Crea una repository su GitHub.
2. Scarica e apri lo ZIP.
3. Carica nella repository tutti i file e le cartelle contenuti dentro `deacon-biamonte-spa`, non la cartella ZIP chiusa.
4. Vai su **Settings > Pages**.
5. In **Build and deployment** scegli:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Salva e aspetta qualche minuto.

## File principali

- `index.html`: contenuti e struttura del sito
- `style.css`: grafica
- `script.js`: menu mobile, animazioni e galleria
- `assets/img`: foto e loghi

## Nota

Il nome "Deacon Biamonte S.p.A." è usato come titolo grafico/personale del sito.
Prima di pubblicare dati personali, foto di altre persone, numero, email e posizione, verifica sempre il consenso degli interessati.
